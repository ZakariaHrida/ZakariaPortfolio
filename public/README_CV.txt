CV FILES SETUP
==============

This folder should contain your CV files in PDF format:

1. cv_en.pdf - Your English resume/CV
2. cv_fr.pdf - Your French resume/CV

CURRENT STATUS:
The current files are placeholder text files. You need to replace them with actual PDF files.

INSTRUCTIONS:
1. Delete the placeholder files (cv_en.pdf and cv_fr.pdf)
2. Add your actual PDF files with the exact same names:
   - cv_en.pdf (English version)
   - cv_fr.pdf (French version)
3. Ensure the files are valid PDFs
4. The download button in the Hero section will automatically work with your files

NOTE:
The CV download functionality works automatically based on the selected language.
When users click "Download CV" in English, it downloads cv_en.pdf
When users click "Télécharger le CV" in French, it downloads cv_fr.pdf
